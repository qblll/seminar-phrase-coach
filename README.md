# Seminar Phrase Coach

A simple responsive Next.js app for international postgraduate students who want more natural English expressions for seminar discussion.

## What is included

- Next.js with TypeScript
- Tailwind CSS
- A single-page coaching interface
- Six seminar situations and three tone choices
- Three mock English suggestions with short explanations
- Copy buttons for each suggestion

This first version does not use login, payments, a database, or an API. The suggestions are mock outputs generated in the browser.

## Run the project

Install dependencies:

```bash
pnpm install
```

Start the development server:

```bash
pnpm dev
```

Then open the local URL printed in the terminal, usually:

```text
http://localhost:3000
```

## Useful checks

```bash
pnpm typecheck
pnpm build
```
